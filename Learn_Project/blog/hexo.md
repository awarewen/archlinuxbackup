# 基本了解

- 可以使用markdown或其他渲染引擎解析文章

# 安装

  1. 需要 
  > Node.js >=10.13
  >> 提供nodejs 内置的npm第三方包管理器
  > Git

- 使用以下命令安装
```
npm install -g hexo-cli
```

# 执行hexo

  1. ```
  npx hexo <command>
  ```

  2. 将hexo 目录下的node_modules 添加到环境变量中可以直接使用hexo <command> 
  ```
  echo 'PATH="$PATH:./node_modules/.bin"' >> ~/.profile
  ```
