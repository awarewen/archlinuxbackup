# 第一节：桶排序

1. 简单的桶排序，核心是利用数组下标对每一个数出现的次数进行计数

```c
  scanf (" %d", &temp);
  bucket[index]++;//核心计数
```
- 缺点是如果要排序的数值过大则会需要更多的桶
    - 例如: `2 2 777 99999 99999`
    - 这里虽然只有5个数,但是却需要`99999`个桶, 在`99999`这个位置做个标记
    - 这并不是真正的桶排序，因为在有人名的情况下该算法就无能为力了

- 算法的时间复杂度为`O(M+N)`

# 冒泡排序

- 每次比较相邻的两个元素，如果顺序错误则调换位置
```c
for (count = 0; count < need_sort; count++)
    for (index = 0; index < need_sort; index++)
        if (bubble[count] < bubble[index]){ //如果前一个数小于后面一个数则调换位置
          temp = bubble[count];
          bubble[count] = bubble[index];
          bubble[index] = temp;
        }
```

- 使用结构体的冒泡可以排序待人名的序列
- 冒泡排序的时间复杂度在`O(N^2)`, 一个非常高的复杂度
